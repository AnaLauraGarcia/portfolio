About Me

Hola! Mi nombre es Ana Laura Garcia. Analista de sistemas en el último período de formación académica, con conocimientos adquiridos en análisis de datos, ciencia de datos, full stack Python, bases de datos, machine learning e interés en el desarrollo de sitios web, programación back end e integración de servicios web. Me apasiona la tecnología digital, el diseño de software y el desarrollo de servicios digitales.

Poseo experiencia en Python, C, PHP, JavaScript, HTML, CSS y SQL, así como en el uso de herramientas como XAMPP, MySQL Workbench, Docker, Visual Studio Code y GitHub.

Soy una persona altamente capaz, con un sentido de pertenencia y una gran responsabilidad.
